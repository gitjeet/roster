

### Quickstart
1.  <b>Start the App</b>:
  - Using Python:
    `pipenv run python asgi.py`

  - Using Docker:
    `docker build -t sqlmodel-api:latest . && docker run -p 8080:8080 sqlmodel-api:latest`

2. <b>Use Openapi at</b>: `http://localhost:8080/#/`

### Semantic Search
